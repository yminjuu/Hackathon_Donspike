## 주요 라이브러리
- 그래프: recharts
- Carousel(회전목마 슬라이드): react-slack
- Datepicker : React Datepicker
- 랜딩페이지 애니메이션: gsap

## 실행
```
git clone https://github.com/DONTSPIKE/DONTSPIKE_FE.git
cd DONTSPIKE_FE

yarn install
yarn run dev
```
